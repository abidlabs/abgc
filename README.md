# ABGC Practice Exam

Interactive practice exam for the American Board of Genetic Counseling (ABGC) certification exam.

## Features

- 93 practice questions covering all exam domains
- Immediate feedback with correct/incorrect highlighting
- Detailed explanations for each question
- Full-screen, centered interface optimized for exam practice
- Keyboard navigation support

## Usage

Open `exam.html` in your web browser to start the practice exam.

## Questions

Questions are stored in `questions.md` and answers/explanations are in `answers.md`.
